# 更新日志

# 1.2.0-BETA

# 版本日志

  1. 修复在创建uuid时不能为null的bug,已剔除该参数。
  2. 修复加解密的自定义方法没有用到方法中的解密次数参数。
  3. 更改为application.yml进行基础配置。
  4. 修正数据源配置，现在直接在application.yml配置druid数据源，支持配置多数据源(将在文档中有所提及)
  5. wiki 帮助文档 迁移到 https://starmcc.github.io/qm-framework
  6. 从该版本开始带有BETA为先行版(尝鲜版,会有较多未知bug)，带有RELEASE为稳定版。

注意：1.2.0 开始不在采用 qm-application.properties 的配置，配置迁移到 application.yml。

# Maven依赖

中央仓库：https://search.maven.org/

阿里云仓库：https://maven.aliyun.com/mvn/search

Maven镜像仓库：https://mvnrepository.com/artifact/com.starmcc

<dependency>
    <groupId>com.starmcc</groupId>
    <artifactId>qm-framework</artifactId>
    <version>1.2.0-BETA</version>
</dependency>
