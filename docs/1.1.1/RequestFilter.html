# 特殊请求过滤

# 介绍

在使用框架时，往往我们在对接第三方的API回调时并不适用我们框架中的版本控制，请求数据规范，响应数据规范。

所以我们需要将这些第三方API回调我们服务端的接口过滤掉。

# 配置

qm-application.properties文件中的配置

#※※※※※※※※※特殊请求过滤※※※※※※※※※
#特殊请求不进行解析(包括版本控制和解析json等)
#该配置主要排除第三方API调用接口时特殊请求而框架自动解析json的问题
#适用于动态配置，例：/user/**
request.special.uri-[0]=

配置多个情况下，我们以[下标]进行多个配置。
