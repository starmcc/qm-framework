# Spring管理器

QmSpringManager

在日常开发中，没有给Spring管理的类中，我们需要获取spring IOC容器中管理的对象，而QmSpringManager就对这一需求进行了友好的实现。

# 使用

在任何场合中，我们都可以使用该管理器获取我们所需的Bean。

public void test (){
    UserService userService = QmSpringManager.getBean(UserService.class);
    userService.xxxxxx
}

在一些特殊的地方我们需要注意：请时刻留意你调用时，是否在工具初始化之前。

请不要在变量初始化时直接调用该管理器

出现如下信息时表示该管理器已初始化成功。
